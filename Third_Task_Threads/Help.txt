Программа получает число N и количество потоков на ввод.
Далее вычисление выражения происходит до N параллельно
во всех потоках.

2 формата ввода данных - из консоли и из файла.

В консоли принимаются на вход 2 целых чиста.

Из файла аналогично: файл должен содержать только 2 целых числа.

Первый java файл - использование ExecutorService.

Второй java файл - ручное создание потоков.
